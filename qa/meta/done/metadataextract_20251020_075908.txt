# SAP SuccessFactors Metadata Extraction Report
## HCMS Program - Queensland Department of Education

### Executive Summary
This comprehensive metadata extraction report provides detailed analysis of the SAP SuccessFactors metadata landscape within the hybrid SAP/Microsoft Azure ecosystem. The extraction focuses on core SuccessFactors components and their integration with downstream systems.

---

## 1. Core SuccessFactors Modules Metadata

### 1.1 Employee Central (EC) Metadata
- **Primary Function**: Core HR master data management
- **Data Model Type**: Object-oriented data model with temporal dependencies
- **Key Objects**: Employee records, organizational units, job roles, job classifications
- **Relationships**: Referential integrity maintained through temporal dependencies
- **Workflows**: Integrated with payroll and external systems via Azure Data Services
- **Business Rules**: Embedded within the object-oriented data model
- **Dependencies**: Critical integration with Employee Central Payroll (ECP)

### 1.2 Payroll Module Metadata
- **Integration Points**: SAP Integration Suite connectivity to Microsoft Fabric
- **Data Flow**: Real-time and batch processing capabilities
- **Temporal Management**: Payroll period-based data synchronization
- **Compliance Requirements**: Queensland Government regulatory compliance

### 1.3 Recruitment Module Metadata
- **Data Objects**: Candidate profiles, job requisitions, application tracking
- **Workflow Integration**: End-to-end recruitment process automation
- **External System Dependencies**: Integration with downstream systems (CIS, IAM, Alloc8)

### 1.4 Onboarding Module Metadata
- **Process Objects**: Onboarding workflows, task assignments, compliance tracking
- **Temporal Dependencies**: Event-driven onboarding triggers
- **Integration Points**: SuccessFactors to external system data synchronization

## 4. Field Definitions and Properties

### 4.1 Core Employee Fields
```
Employee ID (empId):
- Type: String (12 characters)
- Unique Identifier: Primary key for employee records
- Temporal: Non-temporal field
- Security: Restricted access based on role permissions

Employee Name Fields:
- firstName: String, Required, Personal Information
- lastName: String, Required, Personal Information
- displayName: String, Calculated field

Employment Fields:
- employmentType: Picklist, Required, Job Information
- startDate: Date, Required, Temporal
- endDate: Date, Optional, Temporal
```

### 4.2 Payroll-Specific Fields
```
Compensation Fields:
- annualSalary: Decimal, Currency field, Temporal
- hourlyRate: Decimal, Currency field, Temporal
- payGrade: Picklist, Reference to FO PayGrade
- payScale: Picklist, Reference to FO PayScale

Payment Fields:
- bankAccount: String, Encrypted, Personal Information
- taxCode: String, Reference to tax framework
- superannuation: Decimal, Percentage field
```
