# SAP SuccessFactors Onboarding Integration Metadata Extraction Report

## Executive Summary
Metadata extraction performed for HCMS onboarding integration involving iRegister, ServiceNow, and IT equipment provisioning triggers. This analysis covers the metadata required to support automated identity and access management for new starters.

---

## 1. OData API Entity Sets - Onboarding Integration Metadata

### 1.1 Onboarding 2.0 Core Entities
```
Entity Set: OnboardingProcess
- Key Fields: processId, employeeId, startDate, status, workflowId
- Relationships: Employee (EC), User (UM), Position (EC)
- Integration Points: Event Mesh triggers, Workflow service

Entity Set: OnboardingStep
- Key Fields: stepId, processId, stepName, status, completionDate
- Dependencies: iRegister integration step, ServiceNow provisioning step
- Business Rules: Conditional routing based on employee type
```

### 1.2 Identity Management Integration Entities
```
Entity Set: IdentityProvisioningRequest
- Key Fields: requestId, employeeId, identityType, targetSystem, status
- Target Systems: iRegister, Entra ID, SAP IAS/IPS
- Security Context: RBAC roles, MFA requirements, audit trail

Entity Set: IdentityAssignment
- Key Fields: assignmentId, userId, roleId, effectiveDate, expiryDate
- Relationship: User → Role → Permission mapping
- Compliance: IS18:2018 audit requirements
```
